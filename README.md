# 🎬 Movie Search App

A fully responsive Movie Search App built using React JS. This application allows users to search movies dynamically using an API.

## 🚀 Features
- Search movies dynamically using API
- Fully responsive design
- Async/Await for data fetching
- Error handling
- React useEffect hook
- Clean and user-friendly UI

## 🛠️ Technologies Used
- React JS
- JavaScript
- CSS
- Movie API
